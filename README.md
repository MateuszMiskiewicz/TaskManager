# Task Manager API

Simple REST API for managing tasks built with FastAPI.

## Features

* Create task
* Get all tasks
* Get task by id
* Update task
* Delete task

## Tech Stack

* Python
* FastAPI
* SQLAlchemy
* SQLite

## Installation

Clone the repository:

git clone https://github.com/MateuszMiskiewicz/TaskManager.git

Go to project directory:

cd TaskManager

Install dependencies:

pip install -r requirements.txt

Run the server:

uvicorn main:app --reload

## API Documentation

After running the server open:

http://127.0.0.1:8000/docs

Interactive documentation is automatically generated by FastAPI.

## Example Endpoints

Create task

POST /tasks

Get all tasks

GET /tasks

Get task by id

GET /tasks/{task_id}

Update task

PUT /tasks/{task_id}

Delete task

DELETE /tasks/{task_id}
